Overview
Gati Bot is a high-performance, fully autonomous mobile robot (AMR) developed by SVR Robotics. Designed for industrial-grade reliability and high-accuracy positioning, Gati Bot provides a robust hardware and software ecosystem for advanced robotics research, professional development, and commercial automation.

Built with a focus on precision and scalability, the Gati Bot platform integrates seamlessly with the ROS 2 ecosystem, offering a "plug-and-play" experience for complex autonomous workflows.

Key Features
Total Autonomy: Equipped with a native ROS 2-based navigation stack for seamless SLAM (Simultaneous Localization and Mapping) and intelligent path planning.

High-Accuracy Positioning: Leveraging advanced sensor fusion and proprietary calibration from SVR Robotics, Gati Bot delivers centimeter-level localization accuracy even in dynamic environments.

Industrial-Grade Build: Engineered for durability and extended duty cycles, making it a reliable choice for production-level testing and deployment.

Modular Architecture: Easily expandable with custom sensors, actuators, and computing modules to suit specific industrial requirements.
